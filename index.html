Anki add-ons

Anki magnifying glass

Anki magnifying glass adds a keyboard shortcut to Anki that makes the mouse cursor a magnified square of the Anki window at the position of the mouse cursor.

The Anki enumeration tool adds a flashcard making window to Anki which allows for quickly making a group of flashcards that together form an enumeration.

I wrote these while I was a medical student at Quinnipiac University. The magnifying glass was really nice for studying anatomy. I used the enumeration tool to remember things like the steps involved in physical exams (testing all of the cranial nerves for example).

AnkiBooks2

AnkiBooks2 is a Ruby on Rails clone of ankibooks.com. It allows users to register, log in, write books/articles, and download Anki flashcard decks generated from the articles. There is a permissions system where for each book, a user is an owner, editor, or reader. Owners are able to share the book with others, lock articles from being edited, and control the public/private status of the book. Public books can be read without having to log in to the website. Image uploading to articles is also implemented, and the images are included in the downloaded flashcards, however due to Heroku's ephemeral file system this is not supported in the deployed application.

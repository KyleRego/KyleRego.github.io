Anki add-ons

Anki magnifying glass

Anki magnifying glass adds a keyboard shortcut to Anki that makes the mouse cursor a magnified square of the Anki window at the position of the mouse cursor. I would include an image of how this looks except for the fact that the mouse cursor doesn't appear in screenshots but I hope you get the idea.

Anki enumeration tool

The free Anki enumeration tool adds a special flashcard making window to Anki which allows the user to quickly make a group of flashcards where each card constitutes one part of an entire enumeration.

AnkiBooks2

AnkiBooks2 is a clone of ankibooks.com which I developed to learn Ruby on Rails. It is a web application that allows users to register, log in, and write books/articles. The articles are written in Markdown in the editor and rendered as HTML in the reader. The fun thing is that there is a supported syntax extension to Markdown which is used to specify certain sentences of the article as Anki cloze deletion notes. The articles are then downloadable as decks of Anki flashcards.
